Sincronizar com Celular via Wi-Fi
O servidor local iniciado pelo INICIAR_ALBUM.bat permite sincronização automática.
Acesse o álbum pelo celular no mesmo Wi-Fi e os dados serão compartilhados via arquivo JSON.
Passo a passo:
1. Clique duas vezes em INICIAR_ALBUM.bat no seu PC
2. Anote o endereço IP exibido (ex: 192.168.1.10:8765)
3. No celular (mesmo Wi-Fi), abra o Chrome e acesse http://IP:8765/celular.html
4. Use o botão "Exportar JSON" no PC e "Importar JSON" no celular para sincronizar
5. O arquivo album_copa_2026.json pode ser enviado por WhatsApp, Drive, e-mail etc.
